# Project Charter

## プロジェクト名

Influencer Discovery & Deep Analysis Platform
（インフルエンサー候補抽出・深掘り評価プラットフォーム）

## 運営会社

DXAI Solutions

## プロジェクト目的

インフルエンサーマーケティングに関わる店舗・代理店・営業担当が、インフルエンサー候補を効率的に発見・評価・比較し、データに基づいた意思決定ができるプラットフォームを提供する。

## MVP スコープ

1. Instagram URL または username でプロフィールを深掘り分析
2. ブランド適合スコア / リスクスコア / 疑似アクティブ度スコアの算出
3. 候補者の保存・比較（最大 10 件同時比較）
4. PDF レポートおよび CSV エクスポート
5. 基本的な認証・RBAC（admin / analyst / viewer）

## 非目標（MVP では行わない）

- TikTok の大規模自動探索
- SNS への自動投稿・DM 送信
- Google Maps / MEO 操作
- リアルタイムクローリング（非承認スクレイピング）
- フォロワー購入支援

## 最重要制約

1. **公式経路優先**: Instagram Graph API（公式）のみ使用
2. **データ分離**: 事実/推定/LLM生成コメントをフィールドレベルで区別
3. **欠損明示**: 取得できないデータは `status: "unavailable"` を返す
4. **断定禁止**: 推定値には根拠と信頼度（low/medium/high）を必ず付ける
5. **TikTok 制約**: MVP では本人連携・手動入力のみ

## ドメイン用語集

| 用語 | 定義 |
|------|------|
| インフルエンサー | フォロワーに影響を与える SNS アカウント |
| エンゲージメント率 | (いいね+コメント+保存) / フォロワー数 × 100 |
| 疑似フォロワー | 購入・ボットによる偽フォロワー（直接確認不可、推定のみ） |
| ナノインフルエンサー | フォロワー 1K〜10K |
| マイクロインフルエンサー | フォロワー 10K〜100K |
| スナップショット | 特定時点のプロフィールデータの保存 |
| ブランド適合スコア | LLM が評価する、ブランドとの適合度（0〜100） |
| リスクスコア | 依頼に伴うリスクの推定値（0〜100、高い=リスク高） |
| 疑似アクティブ度スコア | 実際のエンゲージメントの質の推定（0〜100） |

## 想定ユーザー

- 店舗運営者
- 小規模代理店
- インフルエンサーマーケ支援会社
- 営業担当
- アナリスト
- 管理者（システム管理）
