# ユーザーストーリーテンプレート

## 形式

```
As a [ユーザーロール],
I want to [操作・機能],
so that [得られる価値・目的].
```

## 例（このプロジェクト固有）

### 候補検索・追加

```
As an analyst,
I want to enter an Instagram URL or username to analyze an influencer profile,
so that I can quickly evaluate the influencer without manual data collection.
```

### スコア確認

```
As an analyst,
I want to see the brand fit score with its basis and confidence level,
so that I can make informed decisions about which influencers to approach.
```

### 比較

```
As a marketing manager,
I want to compare up to 10 influencer candidates side by side,
so that I can select the best fit for my campaign.
```

### レポート出力

```
As a sales person,
I want to export a PDF report of influencer analysis,
so that I can share the findings with my clients in a professional format.
```

## ユーザーストーリーの品質基準

- [ ] ロールが具体的か（「ユーザー」は NG）
- [ ] 操作が明確か（「使いたい」は NG）
- [ ] 価値が具体的か（「便利に」は NG）
- [ ] テスト可能か（受け入れ条件が書けるか）
- [ ] MVP スコープ内か
