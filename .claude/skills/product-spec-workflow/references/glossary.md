# 用語集

## プラットフォーム固有用語

| 用語 | 定義 |
|------|------|
| 候補（Candidate） | 分析・保存対象のインフルエンサーアカウント |
| 深掘り分析 | URL/username を起点にスコアリングまで行う分析フロー |
| スナップショット | 特定時点の Instagram プロフィールデータの保存 |
| データ種別 | fact（事実）/ estimated（推定）/ llm_generated（AI生成）の分類 |
| 信頼度 | low / medium / high — 推定データの確実性レベル |

## Instagram 関連

| 用語 | 定義 |
|------|------|
| Graph API | Meta が公式提供する Instagram データ取得 API |
| ユーザートークン | インスタグラムユーザーの認証トークン（有効期限 60 日） |
| ビジネスアカウント | インサイト取得に必要なアカウント種別 |
| 非公開アカウント | フォロワー以外にはデータ非公開のアカウント |

## スコアリング関連

| 用語 | 定義 |
|------|------|
| ブランド適合スコア | ブランドとインフルエンサーの適合度（0〜100、LLM生成） |
| リスクスコア | 依頼に伴うリスク度（0〜100、高い=リスク高、推定） |
| エンゲージメント品質スコア | フォロワーとの関係の質（0〜100、計算値） |
| 疑似アクティブ度スコア | 本物のエンゲージメントの割合推定（0〜100、推定） |
| 総合スコア | 4つのサブスコアを重み付け合算したスコア（0〜100） |

## ユーザーロール

| ロール | 権限 |
|--------|------|
| admin | 全操作 + ユーザー管理 |
| analyst | 候補追加・分析・レポート出力 |
| viewer | 閲覧のみ（追加・削除不可） |
